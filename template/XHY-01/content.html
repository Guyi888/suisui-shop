    辅助类型详细内容

    安卓手游辅助类型

    1. 一体直装

    需要卸载原来的正版游戏,只下载网盘修改过的游戏,然后直接登入游戏就行,自带辅助外挂

    2. 容器/分身直装

    使用官方正版游戏，只需要下载他的容器安装包，类似于分身应用，然后把你的游戏添加进容器里面，从容器里面启动游戏即可

    3. Root插件

    使用官方正版游戏和MT管理器工具，打开游戏后给予插件应用ROOT权限直接使用或者是在MT管理器ROOT执行插件使用即可

    4. Root内核

    使用官方正版游戏和MT管理器工具，在MT管理器ROOT执行驱动后，再ROOT执行内核辅助启动使用即可

    • 通用教程详细内容

    新手小白下载保存指引

    下载项目教程：

    1. 手动复制你想要下单的商品介绍的下载链接（若无法复制请在手机自带浏览器打开本站）
    2. 打开手机自带浏览器新页面，顶部粘贴链接访问（注意是输入链接的地方，不是搜索栏）
    3. 访问后查看该项目使用的是哪款网盘，去手机应用商店下载对应的网盘app（当前依123云盘为例）
    4. 下载安装好123网盘app注册登陆上，回到浏览器点击在123网盘app中打开，此时跳转回123网盘app
    5. 选择好项目保存到本地网盘，下载至本地即可，123云盘默认保存的路径是/storage/emulated/0/123云盘/

    新手小白打开方法指引

    打开项目教程：

    1. 浏览器访问链接mt2.cn下载mt管理器（一款文件管理器app）安装打开给权限
    2. 有ROOT的给ROOT权限，没有的不给，找到对应网盘的下载路径打开，即可找到下载保存的项目
    3. 第一次用请在哔哩哔哩搜索“MT管理器基础教学”有使用教程帮助你快速上手，可以用于后续的直装安装、改文件操作、插件使用、内核驱动刷入等等

    安卓手游辅助使用教程

    一体直装使用教程：

    1. 卸载正版游戏，下载网盘里面修改过的一体安装包
    2. 安装好以后先不要启动修改过的游戏
    3. 手动去手机设置给直装app所有权限(浮窗和储存权限是必须要给的，最好手动给所有权限)
    4. 然后再启动修改过的游戏，输入卡密登录使用即可
    5. qq/vx登陆页面选择扫码登陆即可进入

    安卓容器/分身直装使用教程：

    1. 不用卸载正版游戏，就准备好官方正版游戏
    2. 下载好网盘里的容器安装包，安装好后
    3. 手动去手机设置把容器app的权限全部打开(浮窗和储存权限是必须要给的，最好手动给所有权限)
    4. 然后打开容器，输入激活码登录卡密
    5. 再把你要玩的游戏放入容器当中，类似于双开应用
    6. 然后在容器中打开游戏
    7. qq/vx登陆页面选择扫码登陆即可进入

    安卓root插件使用教程：

    （一）插件是app形式：

    1. 打开你的Root管理器给这个插件app的Root权限
    2. 额外前往手机设置找到这个app给予所有权限
    3. 打开插件输入卡密登入搭配游戏使用即可

    （二）插件是sh文件形式：

    1. 找到下载到手的文件，直接在mt管理器找到
    2. 移动到手机根目录/data的文件夹下
    3. 长按文件点击权限全部打勾（给文件777权限）
    4. 然后点击文件Root执行即可搭配游戏使用

    安卓root内核使用教程：

    1. 找到下载到手的文件，直接在mt管理器找到
    2. 如果下载到手的是zip后缀结尾的压缩包那就先解压
    3. 一般由驱动和启动器两部分，根据自己手机内核版本先Root执行刷入相应版本号的驱动
    4. 刷入成功后，找到启动器文件，把它移动到手机根目录/data的文件夹下
    5. 长按启动器文件点击权限全部打勾（777权限）
    6. 然后点击文件Root执行即可搭配游戏使用

    支付成功后显示未支付：

    1. 下单必须填写正确的下单信息(手机号,QQ号,取卡密码,邮箱号)其中任意一种

    3. 支付完直接回到网站查单页面查询下单信息
    4. 点击立即查询后会弹出卡密信息

    为什么查询订单显示待处理：

    1. 本站的业务订单采用全系统自动处理，下单自动发货
    2. 如果你购买的是卡密类商品，为自动发货，待处理可能是库存不足，平台带有自动补单机制，长时间待处理未发卡密联系客服处理即可
    3. 如果你购买的是充值/DY/KS等业务代刷商品，本站仅仅负责代付提交至机房，一般72H内陆续到账，购前已经说明，漏单/掉量/掉单/吞单均无售后，介意禁拍

    安装app软件报毒失败怎么办：

    1. 先把手机应用宝卸载了(没有请无视)
    2. 去设置里面把'禁止安装恶意软件'关闭
    3. 断网安装
    4. 删除其他辅助软件卸载可能包名重复了导致无法安装
    5. 需要把正版游戏卸载了再进行安装

    辅助刚上来使用就封号：

    1. 辅助封号可搭配防封端口使用用来增长使用时间（最后还是大概率封号）。
    2. 封号已经在【支付页面】【网站教程】【购前必看】【文章必读】等等地方甚至付款之前已经做出说明，本站所有商品均是网站收集，不负责任何封禁甚至秒封同样后果自负概不退换！

    非正版应用/微信签名不一致：

    1. QQ登录提示非正版应用：使用一键上号的上号器或卸载QQ扫码上号
    2. WX登录提示签名不一致：扫码登录即可或扫码VX小号一键上号

    输入卡密提示服务器连接超时：

    1. 软件开发商关闭服务器在更新或维护导致暂时无法登入！
    2. 软件服务器被打导致短时间无法登录，耐心等待恢复！
    3. 若提示卡密不存在请带上网站查询的详细订单截图联系人工客服

    机器码错误/请在原设备登录：

    1. 出现这些情况自己更换了登录设备的设备码导致
    2. 手机刷机和更换虚拟机更换框架都会导致设备码不一样
    3. 桌面和虚拟机的设备码也不一样最常见的就是桌面登录了卡密，然后去虚拟机里面登录，或者更换虚拟机使用，更换框架使用
    4. 有一些直装软件卸载重装也会设备码错误
    5. 卡密绑定设备只会绑定第一次登入的机型
    6. 这种问题先看看软件有没有自带解绑按钮或解绑链接
    7. 如果没有带上订单号问客服能不能解绑，如果不能解绑请自行承担

    国际服提示一串英文代码：

    1. 先打开游戏，中间出现英文的时候，再退出
    2. 使用MT管理器找到下载的obb浏览器下载选择用MT管理器打开，然后随便找个文件夹往里面放一下，找到android进去找到obb然后再找com.tencent.ig，找到obb直接选择移动进去然后重启游戏
    3. 如果还是出现那玩意的话大退一下，如果还是不行，那就是obb下载错了！

    闪退问题解决办法：

    1. 权限都给，不要禁止权限，悬浮窗口别禁止
    2. 手机有任何自带加速游戏的都别开
    3. 重装一次试试，别覆盖安装，卸载干净重启再次安装尝试
    4. 如任何方法都尝试，依旧不行就是不支持你的机型！

    关于sh文件怎么打开操作：

    1. 首先手机需Root权限
    2. 从网盘打开sh文件➯之后mt找到data目录➯手机设置➯关于手机➯详细参数➯内核版本➯找到对应内核版本的驱动
    3. 之后把挂.sh文件解压到data/长按属性/660/600改成777/之后点击文件执行

    苹果免越狱问题以及安装失败解决办法：

    1. 打开网址点击下载如免签名无法下载提示验证不完整无法打开App 则需要自签
    2. 首先要留有足够空间（20G以上）--卸载原先的游戏/删除黑图标--清理浏览器数据--还原所有设置--重新安装
    3. 如果还不行就切换一下网络重新试一遍
    4. 所有问题基本解决！还不行的就点一下签名旁边的"重置"，重新打开安装一遍
    5. 有时候提示卡密不存在并非是卡密错误，切换一下网络登录几乎可以解决百分之八十
    6. 极个别登录不了提示卡密不存在说明服务器正在被打等等他恢复在登陆就好了！
