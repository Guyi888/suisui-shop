# 岁岁云商城

岁岁云商城（suisui-shop）是一套面向自营发卡、商品销售、代刷对接、分站代理、订单管理、财务流水和后台运营场景的 PHP 商城系统。

> ⚠️ **声明**: 本项目仅供学习、研究和二次开发参考。用于生产环境前，请自行完成安全审计、支付风控、数据备份和合规评估。

## ✨ 功能特性

- **商品商城** — 支持普通商品、发卡商品、分类展示、库存、价格、成本和排序管理。
- **自动对接同步** — 支持同系统/代刷对接，自动同步分类、商品、价格、描述、库存和上下架状态。
- **同步去重修复** — 对接商品按供应商商品 ID 去重，避免同一商品重复入库导致数量膨胀。
- **商品动态** — 前台 `/toollogs.php` 展示新上架和下架日志，后台可管理商品动态。
- **分站体系** — 支持普通用户、普及版分站、专业版分站，用户列表统一展示所有用户，分站列表专注分站管理。
- **站点任务** — 支持充值、下单、销售额、邀请、签到、推广商品等任务，达标后可自动派发奖励。
- **充值返利** — 支持满额返利、阶梯比例、在线充值返利、加款卡返利和后台手动加款返利。
- **财务流水** — 收支明细按用户、类型、时间、关键词筛选，并展示收入/支出汇总。
- **每日签到** — 支持签到奖励、连续签到，并在后台查看最近签到用户记录。
- **模板与自定义样式** — 首页模板统一适配站点数据，支持前台自定义 CSS 注入。
- **后台运营 UI** — 商品、分类、订单、价格、消息、工单、更新日志、联系赞助等后台页面持续优化。

## 🚀 快速开始

### 环境要求

- PHP 7.4 或更高版本
- MySQL 5.7+ 或 MariaDB
- Nginx / Apache
- PHP 扩展: `pdo_mysql`、`curl`、`mbstring`、`json`

### 1. 克隆项目

```bash
git clone https://github.com/Guyi888/suisui-shop.git
cd suisui-shop
```

### 2. 上传到站点目录

将项目文件上传到 Web 根目录，例如:

```text
/www/wwwroot/example.com
```

确保运行目录具有写入缓存、配置和上传文件的权限。

### 3. 执行安装

浏览器访问:

```text
https://你的域名/install/
```

按安装向导填写数据库地址、数据库名、用户名、密码和表前缀。

安装脚本会自动创建基础表和运行时功能表，包括:

- `sitetask_log`
- `toollogs_offline`
- `sync_category_map`
- 商品、订单、用户、分类、配置等核心业务表

### 4. 配置站点

安装完成后进入后台:

```text
https://你的域名/admin/
```

建议优先配置:

- 网站基础信息
- 支付接口
- 商品分类与商品
- 自动同步设置
- 充值返利设置
- 首页模板与自定义样式

### 5. 配置自动同步

在后台添加对接站点后，配置自动同步规则。同步入口会自动完成:

- 分类父子关系映射
- 商品新增和更新
- 重复商品去重
- 上架/下架动态写入
- 下架商品处理

建议通过计划任务定时访问同步入口，并保管好监控密钥。
