# 更新日志

## v1.0.0.0 - 岁岁云商城首个可运营重构版本

发布日期：2026-05-19

### 重要说明

- 本版本是岁岁云商城独立运营版的首个正式记录版本，版本号从 `v1.0.0.0` 开始。
- 本次更新聚焦本地品牌资源统一、后台登录页清理、第一批后台运营页面重构、乱码修复和线上同步验证。
- 后续版本规则：超大更新提升第二段，大更新提升第三段，小更新提升第四段。

### 修复问题

- 修复前台、后台 favicon 未统一为本地 Q8 图标的问题，补齐 `favicon.ico` 与 `assets/img/favicon/favicon.ico`。
- 修复后台登录页残留远程背景图片外链、旧模板注释和旧文字说明的问题。
- 修复自动转账接口旧硬编码远程地址，改为后台配置项 `transfer_api_url` 并增加基础 URL 校验。
- 修复 `admin/ajax.php` 中上传文章图片、favicon 上传、卡密详情和首页统计附近的乱码注释、断裂输出与潜在转义问题。
- 修复 `includes/function.php` 分类下拉输出乱码与 HTML 转义问题，分类名称统一通过 `htmlspecialchars` 输出。
- 修复用户中心头部禁用/到期提示、分站加价模板说明、商品列表注释中的乱码，减少前台可见异常字符。
- 清理更新记录文件本身的历史乱码，后台 `admin/changelog.php` 改为实体输出，避免页面再次出现编码断裂。

### 新增功能

- 新增本地 Q8 logo 资源，后台登录页右侧改为本地图片展示。
- 新增后台运营页面通用 shell 样式，提供统一统计卡、工具栏、批量操作、空状态和移动端布局能力。
- 新增 README 首页“最近更新”展示，方便 GitHub 首页直接看到最新版本摘要。
- 新增后台更新日志页面的版本记录说明，与 GitHub 更新记录同步维护。

### 优化改进

- 后台公共头部接入新的 admin shell 资源版本，统一品牌、favicon helper 与页面基础视觉结构。
- 重构 `shoprank.php` 商品销量排行界面，补充统计、搜索和快捷入口。
- 重构 `fakalist.php` 发卡库存界面，补充库存统计、批量操作、状态切换和确认交互。
- 重构 `shopnoo.php` 商品介绍批量替换界面，补充预览和执行流程。
- 重构秒杀商品管理界面，包含 `seckill.php`、`seckill-table.php`、`assets/js/seckill.js`、`assets/js/seckilledit.js`。
- 重构 `kmlist.php` 加款卡密界面，补充生成结果页、金额统计、搜索弹窗、清空确认页和删除确认事件。
- 重构 `support.php` 联系与赞助界面，改为统一维护信息页并清理历史乱码。

### 界面样式

- 后台页面统一向分类列表、商品列表的新后台 shell 风格靠齐。
- 登录页视觉资源统一走本地 logo 与 CSS 渐变，不再加载远程背景图片。
- 秒杀商品管理移除旧内联点击事件，中文 JS 提示改为 Unicode 转义。
- 业务页面图标继续统一使用 FontAwesome，不引入 glyphicon。
