# 更新日志

## v1.0.2.3 - 浮动客服配置

发布日期：2026-05-21

### 重要说明

- 本版本将前台客服入口调整为可配置的全站浮动客服，站长可按需选择跳转链接、QQ 对话、Chatwoot 或自定义代码。
- 原内置客服窗口入口已移除，避免继续维护旧的前台轮询聊天窗口、上传和提示音逻辑。

### 修复问题

- 移除后台公共头部中旧客服消息轮询和提示音引用，避免后台继续请求旧客服资源。
- 删除旧内置客服提示音资源，减少无用静态文件残留。

### 新增功能

- 新增后台“浮动客服配置”入口。
- 支持配置客服开关、客服方式、按钮文字、FontAwesome 图标、按钮颜色、文字颜色、显示位置、边缘距离、手机端显示和新窗口打开。
- 支持 Chatwoot 地址与 Website Token 配置。
- 支持站长粘贴可信第三方客服自定义代码。

### 优化改进

- 客服入口统一由 `template/default/chat/widget.php` 输出，避免每个模板单独硬编码。
- 左侧菜单在“内容与外观”下新增“浮动客服配置”，便于站长直接管理。

### 界面样式

- 浮动客服按钮适配 PC 和手机端，手机端默认只显示图标按钮。
- 按钮图标统一使用 FontAwesome。

### 部署说明

- 程序缓存版本更新到 `2026052106`。
- 覆盖更新后可在后台“内容与外观 / 浮动客服配置”中开启并配置客服入口。
- 本版本不包含数据库结构变更。

### 后续计划

- 继续按模板检查浮动按钮与页面底部导航、购物车入口等固定元素的遮挡情况。
- 根据实际运营反馈补充更多客服类型或模板级样式适配。
