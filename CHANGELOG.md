发布日期：2026-05-23

- 分页链接会保留当前明细类型，翻页不会丢失卡片筛选状态。

### 界面样式

- 收支明细新增紧凑型经营卡片样式，并迁移到后台公共 shell 样式文件中维护。
- 移动端卡片改为双列展示，明细面板按钮和筛选栏在小屏幕下自动占满宽度。

## v1.0.2.15 - 同步商品来源保护与价格展示修复
发布日期：2026-05-23

### 重要说明

- 本版本面向岁岁云商城独立运营版本，重点修复自动同步在删除规则开启时可能误清理手动商品的问题。
- 同步删除现在只处理由自动同步流程登记过来源的商品，手动新增商品不会因为填写了供货站 ID 或商品 ID 而被上游缺失规则误删。
- 本版本会自动创建 `pre_sync_goods_map` 同步来源映射表，用于区分自动同步商品和后台手动商品。

### 修复问题

- 修复同系统自动同步在 `delete_rule=2` 时，仅按 `shequ + goods_id NOT IN 上游列表` 判断，可能删除手动新增商品的问题。
- 修复后台商品列表和价格弹窗在使用加价模板时，展示价格与实际生效价格不一致的问题。
- 修复专业版分站利润计算在存在站长自设成本价时，仍可能回退到默认成本字段的问题。

### 新增功能

- 新增同步商品来源映射表 `pre_sync_goods_map`，自动记录供货站商品 ID 与本地商品 ID 的对应关系。
- 自动同步新增或更新商品时会维护来源映射，后续缺失处理只对已确认来源的同步商品生效。

### 优化改进

- 同步商品缺失处理从直接扫描商品表改为通过来源映射表筛选，降低误删运营自建商品的风险。
- 删除同步商品时同步清理来源映射，保持后续同步判断干净。
- 商品价格展示统一按加价模板计算后输出，便于后台核对销售价、普及版价格和专业版价格。

### 界面样式

- 本版本不调整前台或后台页面样式，仅修正后台商品价格展示数据。

### 部署说明

- 程序缓存版本更新到 `2026052302`。
- 覆盖更新后首次运行自动同步会自动创建 `pre_sync_goods_map` 表，无需手动执行 SQL。
- 建议更新后在后台自动同步中执行一次同步，确认手动商品不会因上游缺失规则被删除，同时检查正常上游商品仍可新增、更新和按规则下架。

### 后续计划

- 继续完善同步任务日志，让删除、下架和跳过保护的商品有更清晰的运营可追踪记录。
- 继续整理商品同步配置页的风险提示，帮助站长理解“下架/删除上游缺失商品”的影响范围。

### 重要说明

- 已存在的对接商品在再次同步时可按上游字段修正 `multi` 与 `goods_type`，不需要手动逐个编辑商品。

### 界面样式

- 本版本不调整前台或后台页面样式。

### 部署说明

- 程序缓存版本更新到 `2026052215`。
- 本版本不包含数据库结构变更。
- 覆盖更新后建议在后台自动同步中对同系统上游执行一次“仅修复已有商品”或完整同步，检查发卡商品、支持多份购买商品的 `multi/goods_type/min/max` 是否与上游一致。

### 后续计划

## v1.0.2.12 - 自动同步任务与分站价格模板修复

发布日期：2026-05-22

### 重要说明

- 本版本面向岁岁云商城独立运营版本，稳定自动同步立即执行链路，并修复分站用户中心价格模板与批量调价的实际运营问题。
- 分站商品价格规则调整为：默认所有商品跟随当前站点价格模板；存在商品自定义价格时优先使用自定义价格；点击恢复价格会删除自定义价格并回到当前模板价格。
- 本版本不包含数据库结构变更，不会在客户站更新时自动删除旧价格数据。

### 修复问题

- 修复用户中心站点管理中加价模板设置后，商品列表部分价格仍不按模板计算的问题。
- 修复用户中心商品列表批量修改“下级专业版价格”等字段时，保存参数未完整写入导致修改不生效的问题。
- 修复部分历史自定义价格低于当前站长成本时，分站商品列表可能显示异常低价的问题。
- 修复后台“立即执行同步”在大量商品场景下可能因浏览器或网关等待超时，被误判为同步失败的问题。
- 修复自动同步在远端商品字段名不一致时，可能写入空商品名或重复触发任务的问题。

- 自动同步立即执行改为后台任务模式，页面发起后可通过任务状态查看执行进度、完成数量、失败数量和错误摘要。

- 分站价格计算新增站点加价模板兜底逻辑，没有自定义价格的商品会实时跟随当前模板。

### 优化改进

- 分站价格计算保留站长自身成本口径，模板加价只影响下级和销售展示价格，避免成本列被模板误抬高。
- 商品自定义价格写入时增加层级保护，避免下级专业版、下级普通版和销售价低于应有成本。
- 已存在商品本地名称为空时，会在远端返回有效名称后自动修正；远端名称仍为空的新商品会跳过写入。
- 同步锁释放改为异常保护，减少同步中断后任务锁残留的概率。

### 界面样式

- 自动同步设置页补充后台任务面板、进度状态和结果摘要，保持新后台 shell 风格。
- 用户中心商品列表继续沿用现有表格与批量操作交互，本次只修复价格计算和保存链路。

### 部署说明

- 程序缓存版本更新到 `2026052212`。
- 本版本不包含数据库结构变更。
- 覆盖更新后建议刷新后台自动同步设置页，并在用户中心检查站点价格模板、商品批量改价和恢复价格功能。
- 如旧站存在大量历史自定义价格，更新代码不会自动删除；需要回到模板价格的商品可在用户中心使用“恢复价格”处理。

### 后续计划
