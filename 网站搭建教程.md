# 岁岁云商城系统搭建教程

## 一、环境要求

- 操作系统：推荐 Linux（CentOS 7+/Ubuntu 18+），Windows 也可测试使用。
- Web 服务：Nginx 或 Apache。
- PHP 版本：推荐 PHP 7.4，后续 PHP 8 兼容请以更新日志说明为准。
- 数据库：MySQL 5.7+ 或 MariaDB。
- 常用扩展：pdo、pdo_mysql、openssl、fileinfo、mbstring、curl、gd、zip。

## 二、准备工作

1. 下载 GitHub 最新源码包并上传到网站根目录。
2. 配置域名解析，确认域名已指向服务器。
3. 在宝塔中创建站点和数据库。
4. 设置网站目录权限，确保 PHP 可写入缓存、上传和安装锁文件。

## 三、安装步骤

1. 浏览器访问 `http://你的域名/install/` 进入安装页面。
2. 按提示填写数据库信息和管理员账号。
3. 安装完成后确认 `install/install.lock` 已生成。
4. 进入后台：`http://你的域名/admin/`。

## 四、更新提醒

老站更新不要重新运行安装向导。覆盖更新前请先阅读 `UPDATE.md`，并备份网站目录和数据库。

## 五、常见问题

- 数据库连接失败：检查数据库账号、密码、库名、主机和端口。
- 页面空白或报错：检查 PHP 版本、扩展和错误日志。
- 伪静态问题：确认 Nginx/Apache 规则与当前站点配置一致。
- Cloudflare 526：源站 SSL 证书无效，需安装有效源站证书或调整 SSL 模式。
