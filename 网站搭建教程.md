# 岁岁云商城系统搭建教程

## 一、环境要求

- 操作系统：推荐 Linux（CentOS 7+/Ubuntu 18+），Windows 也可
- Web服务器：Nginx 或 Apache
- PHP 版本：7.4
- 数据库：MySQL 5.7 及以上
- 虚拟主机（服务器不用）需要的依赖扩展：pdo、pdo\_mysql、openssl、fileinfo、mbstring、curl、gd、zip
- 依赖一般都已包含在 PHP 安装包中，无需额外安装，但是除了虚拟主机可能缺少一些扩展，需要手动安装。

## 二、准备工作

1. 获取完整源码包，上传至服务器网站根目录
2. 配置好域名解析，确保域名已指向服务器
3. 设置好网站目录权限，确保 PHP 有读写权限
4. 导入源码内的数据库

## 三、安装步骤

1. 浏览器访问 `http://你的域名/install/` 进入安装界面
2. 按提示填写数据库信息、管理员账号等
3. 安装完成后，删除 `/install` 目录或创建 `install/install.lock` 文件
4. 进入后台管理：`http://你的域名/admin/`

## 四、常见问题

- 数据库连接失败：请检查数据库账号、密码、主机、端口是否正确，MySQL 是否已启动
- 页面空白或报错：请开启 PHP 错误显示，检查扩展是否齐全
- 伪静态/重写：如用 Nginx/Apache，需配置伪静态规则
